# Retail EDA Visualization

## Objective
Explore customer spending patterns using Python visualization libraries.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly

## Tasks Performed
- Created synthetic retail dataset
- Built bar chart for category-wise average spend
- Created scatter plot for income vs spend
- Developed interactive and static visualizations
- Added observations for each analysis

## Key Insights
- Spending varies by category
- No strong relationship between income and spend
- Interactive plots help explore patterns

## Project Type
Exploratory Data Analysis (EDA)
